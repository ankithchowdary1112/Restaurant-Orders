# Restaurant-Orders
🔹 Project Overview

This project analyzes restaurant order data across four cuisine categories: American, Italian, Mexican, and Asian.

Using SQL, the dataset is explored to generate business insights such as revenue trends, popular dishes, category performance, and peak ordering hours. The goal is to simulate real-world restaurant analytics using relational database concepts.

🎯 Business Objectives

Calculate total revenue and total orders

Identify top-selling dishes

Compare revenue performance across cuisine categories

Analyze peak ordering times

Determine customer ordering trends
